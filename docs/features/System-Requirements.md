# 動作環境 (System Requirements)

## 1. 開発・実行環境
本プロジェクトは、以下のソフトウェア環境を前提としています。

- **Java**: JDK 21 以上 (LTS)
- **ビルドツール**: Maven 3.9 以上
- **データベース**:
  - MongoDB (ゲーム状態の永続化)
  - H2 Database (ワールド情報、開発用インメモリDB)
- **コンテナ**: Docker / Docker Compose (MongoDB 等のミドルウェア実行用)

## 2. 推奨スペック
マイクロサービス構成であるため、ローカル開発時には複数の JVM プロセスが起動します。

- **メモリ**: 8GB 以上 (16GB 推奨)
- **ストレージ**: 2GB 以上の空き容量

## 3. 技術的制約
- **ネットワーク**: サービス間通信に HTTP/REST を使用するため、内部ポートの開放が必要です。
- **OS**: Windows, macOS, Linux (Java 21 が動作する環境)
