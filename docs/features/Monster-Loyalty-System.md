# モンスター忠誠度システム (Monster Loyalty System)

## 1. 概要
本ドキュメントは、プレイヤーと「仲間のモンスター」との間の絆を示す「忠誠度（懐き具合）」システムの仕様を定義します。忠誠度は、モンスターの戦闘中の挙動、ステータス、および進化に影響を与える重要なパラメータです。

## 2. 忠誠度の基本仕様
忠誠度は各モンスター個体（`MonsterInstanceDomain`）ごとに保持される数値データです。

- **値の範囲**: 0 〜 255
- **初期値**:
    - **捕獲直後**: 50（野生の状態から少し心を開いた状態）。
    - **孵化直後**: 120（生まれた時からプレイヤーを親と認識している状態）。

## 3. 忠誠度の増減メカニズム
プレイヤーとの行動やアイテムの使用によって数値が変動します。

### 3.1 増加条件
| アクション | 変動値 | 備考 |
| :--- | :---: | :--- |
| **同行して歩く** | +1 | プレイヤーと同行した状態で 100 歩 (`subStep`) 経過。 |
| **レベルアップ** | +5 | 戦闘またはアイテムによるレベル上昇時。 |
| **戦闘に勝利** | +1 | 敵モンスターを撃破した際に、戦闘に参加していた場合。 |
| **好物アイテムの使用** | +10 〜 +30 | 「懐きクッキー」など、特定のアイテムを使用。 |

### 3.2 減少条件
| アクション | 変動値 | 備考 |
| :--- | :---: | :--- |
| **戦闘不能（HP 0）** | -20 | 戦闘で倒され、変身解除または拠点へ戻された場合。 |
| **長期間の放置** | -1 / 日 | 拠点や倉庫に預けたまま、現実時間で 24 時間経過。 |
| **嫌いなアイテムの使用** | -10 | 苦い薬など、モンスターが嫌がるアイテムを無理に使用。 |

## 4. 忠誠度の影響

### 4.1 戦闘中の挙動
忠誠度が極端に低い場合、モンスターがプレイヤーの指示を無視することがあります。

- **忠誠度 0 〜 30**: 25% の確率で行動を拒否し、そのターンは「何もしない（Wait）」になります。
- **忠誠度 31 〜 50**: 10% の確率で行動を拒否します。
- **忠誠度 51 以上**: 常に指示に従います。

### 4.2 ステータス補正（絆ボーナス）
忠誠度が高い個体は、プレイヤーへの信頼によって能力が向上します。

- **忠誠度 200 〜 250**: すべての基本ステータス（HP, MP 除く）に **+3%** のボーナス。
- **忠誠度 251 〜 255**: すべての基本ステータスに **+5%** のボーナス、およびクリティカル率に **+5%** の固定ボーナス。

### 4.3 進化への影響
一部のモンスター種族において、進化の条件として「一定以上の忠誠度」が必要となります。
詳細は [モンスター進化システム](./Monster-Evolution-System.md) を参照してください。

## 5. 忠誠度の確認方法
- **ステータス画面**: モンスターの詳細画面で、数値に応じたテキスト（例：「とても懐いている」「少し警戒している」等）が表示されます。
- **拠点の NPC**: 「懐き具合」を診断してくれる NPC が、正確な数値を教えてくれる場合があります。

## 6. データ構造
`MonsterInstanceDomain` に以下のフィールドを追加します。

- `loyalty`: Integer（0 〜 255）

## 7. 今後の拡張
- **親愛スキル**: 忠誠度が最大値に達した時のみ習得・発動可能な強力なスキル。
- **特殊演出**: 忠誠度が高いと、戦闘中にプレイヤーをかばう、あるいは状態異常を自力で治す等の特殊演出が発生する。
