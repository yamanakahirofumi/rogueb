# モンスター特性システム (Monster Trait System)

## 1. 概要
本ドキュメントは、モンスターが種族固有、あるいは個体ごとに保持するパッシブ能力「特性（Trait）」の仕様を定義します。特性は、戦闘中のステータス補正や、環境・状態異常に対する耐性、特殊な行動ロジックなどを提供します。

## 2. 特性の基本仕様
特性は、スキル（能動的なアクション）とは異なり、条件を満たしている間、あるいは常に自動で効果を発揮します。

- **保持場所**:
    - `MonsterDomain.traits`: その種族が標準で持っている特性。
    - `MonsterInstanceDomain.traits`: その個体が現在保持している特性。
- **重複設定**: 同一の特性を複数保持することはできません。効果が重複する場合は、より強力な方が優先されるか、あるいは加算されるかが各特性ごとに定義されます。

## 3. 主要特性リスト
システムで標準的に定義される特性の ID と効果の一覧です。

| 特性 ID | 名称 | 効果概要 | 備考 |
| :--- | :--- | :--- | :--- |
| `REGENERATION` | 自己再生 | 自身のターン終了時、最大 HP の 3% を回復する。 | 自然回復とは別に発動。 |
| `MIASMA_RESISTANCE` | 瘴気耐性 | 環境効果「瘴気」によるダメージおよび自然回復無効化を無視する。 | ランク A 以上のダンジョンで有効。 |
| `FIRE_IMMUNITY` | 火属性無効 | 火属性によるダメージを無効化する。 | |
| `BRACE` | 食いしばり | HP 2 以上の時に致死ダメージを受けても、10% の確率で HP 1 で耐える。 | 忠誠度による「食いしばり」と重複可能。 |
| `SLIME_BODY` | スライム体質 | 物理ダメージを 20% 軽減するが、火属性ダメージが 1.5 倍になる。 | スライム系の固有特性。 |
| `UNDEAD_SOUL` | 不死の魂 | 状態異常「睡眠」「封印」を無効化する。聖なる攻撃に弱くなる。 | アンデッド系の固有特性。 |
| `FLIGHT` | 飛行 | 「落とし穴」「地雷」などの一部のトラップを無効化する。 | 地形による移動コストも無視。 |
| `AGGRESSIVE` | 闘争本能 | 自身の HP が 30% 以下の時、攻撃力が 1.2 倍になる。 | |
| `SCAVENGER` | スカベンジャー | プレイヤーを撃破した際、追加でゴールドを獲得する確率が上昇する。 | |
| `STATUS_IMMUNITY` | 全状態異常無効 | すべての状態異常（毒、麻痺、睡眠等）にかからなくなる。 | ボスモンスター向けの強力な特性。 |

## 4. 特性の獲得と継承

### 4.1 種族特性
各種族（`MonsterDomain`）は、0 〜 3 つ程度の標準特性を持っています。その種族の個体が生成された際、これらは自動的に `MonsterInstanceDomain.traits` にコピーされます。

### 4.2 突然変異 (Mutation)
[モンスター繁殖システム](./Monster-Breeding-System.md) において、稀に両親が持っていない特性が発現する場合があります。
- **発現確率**: 5% 程度。
- **対象**: そのカテゴリ（Type）に関連する特性、または汎用的な特性から抽選されます。

### 4.3 進化による変化
[モンスター進化システム](./Monster-Evolution-System.md) によって種族が変化した際、特性も進化後の種族のものへと更新されます。ただし、進化前の個体が「突然変異」等で得た固有の特性は、そのまま引き継がれる場合があります。

## 5. 実装時の考慮事項
- **計算順序**: ダメージ計算時、ステータス補正の後に特性による軽減・増加を適用します。詳細は [戦闘システム](./Combat-System.md) を参照してください。
- **表示**: クライアント側のステータス画面において、保持している特性の名称と説明が確認できるようにします。

## 6. 今後の拡張
- **特性強化**: 特定のアイテムを使用して、既存の特性を強化（例：自己再生 I → II）する仕組み。
- **連携特性**: パーティ内に特定の組み合わせの特性を持つモンスターがいる場合に発動するシナジー効果。
