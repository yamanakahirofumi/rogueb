# スキル・魔法システム (Skill & Magic System)

## 1. 概要
本ドキュメントは、プレイヤーおよびモンスターが使用可能な「スキル」および「魔法」の仕様を定義します。スキルと魔法は、戦闘においてダメージを与えるだけでなく、回復や状態異常付与などの特殊な効果をもたらします。

## 2. スキル・魔法のカテゴリ
スキルと魔法は、消費リソースや発動特性によって以下の 3 つに分類されます。

### 2.1 物理スキル (Physical Skills)
- **消費リソース**: スタミナ (Stamina)
- **特徴**: 武器の攻撃力に依存した物理ダメージを与えます。命中判定は物理攻撃のルールに従います。

### 2.2 魔法 (Magic)
- **消費リソース**: MP (Magic Point)
- **特徴**: 魔法攻撃力に依存した魔法ダメージを与えます。属性相性の影響を強く受けます。原則として必中（命中率 100%）です。

### 2.3 バフ・デバフ (Buffs & Debuffs)
- **消費リソース**: MP または スタミナ
- **特徴**: 自己または対象のステータスを一時的に増減させ、あるいは状態異常を付与・解除します。

## 4. 習得と継承

### 4.1 習得方法
- **プレイヤー**: レベルアップ時、または特定の「秘伝書（巻物）」を読むことで習得します。
- **モンスター**: 種族ごとに固有のスキルを初期保有しているほか、レベルアップで新たなスキルを開放する場合があります。

### 4.2 継承 (Breeding)
モンスターの繁殖時、両親が持つスキルの一部を子が継承することがあります。
詳細は [モンスター繁殖システム](./Monster-Breeding-System.md) を参照してください。

## 5. モジュール間連携

1. **PlayerOperations**: プレイヤーからのスキル・魔法使用コマンドを受信。
2. **PlayerOperations -> BookOfAdventure**: 使用コスト（MP/Stamina）が不足していないか確認。
3. **PlayerOperations -> CombatSystem (Logic)**: ダメージ・効果計算を実行。
4. **PlayerOperations -> BookOfAdventure / Dungeon**: ステータス更新（コスト消費、HP減少、状態異常付与）を要求。

## 6. 今後の拡張
- **連携スキル**: 特定の条件下で複数のキャラクターが同時に発動する強力なスキル。
- **設置型魔法**: 指定したマスに一定ターンの間効果を発揮する魔法（例: 炎の壁）。
- **スキルツリー**: プレイヤーが自由に成長方針を選択できる仕組み。
