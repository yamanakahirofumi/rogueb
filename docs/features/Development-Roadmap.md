# 開発ロードマップ (Development Roadmap)

## 1. 開発フェーズ

### フェーズ 1: MVP (Minimum Viable Product) - **進行中**
- 基盤となるマイクロサービス群の構築 (Dungeon, World, Objects, BookOfAdventure, PlayerOperations)
- 基本的な 8 方向移動、アイテム拾得、階段昇降の実装
- MongoDB による状態永続化

### フェーズ 2: ゲーム性の向上 - **計画中**
- アイテム識別システムの導入
- ダンジョン生成アルゴリズムの多様化
- 戦闘システムの基礎実装 (攻撃、ダメージ計算、HP管理)

### フェーズ 3: 拡張性と調整 - **検討中**
- 特殊な地形効果やトラップの追加
- モンスター AI の強化
- バランス調整とプレイ感覚（手触り）の向上

## 2. 優先順位
1. データ整合性の確保（セーブデータの信頼性向上）
2. アイテム識別システムの詳細設計と実装
3. UI/UX のプロトタイプ作成
