# ゲーム機能概要 (Game Features)

## 1. コンセプト
`rogueb` は、伝統的なローグライクゲームの要素をモダンなマイクロサービスアーキテクチャで再構築したプロジェクトです。プレイヤーは生成されるダンジョンを探索し、アイテムを収集し、キャラクターを成長させながら深層を目指します。

## 2. 基本操作
プレイヤーは以下の基本アクションを実行できます。

- **移動**: 8方向（上下左右および斜め）への移動が可能です。
- **アイテム拾得**: 足元にあるアイテムや金を拾うことができます。
- **階段の利用**: 上り階段および下り階段を使用してフロア間を移動します。
- **周辺確認**: 現在のフロアの視界内にある地形やエンティティを確認できます。

## 3. 主要なゲームサイクル
1. **ダンジョン入場**: ワールドから探索するダンジョンを選択し、初期位置に配置されます。
2. **探索と行動**: 移動やアイテム拾得を繰り返し、フロアの構造を把握します。
3. **リソース管理**: 拾得したアイテム（武器、防具、指輪、巻物等）を駆使して生存率を高めます。
4. **階層の進行**: 階段を見つけ、より深い（あるいは浅い）階層へと進みます。
5. **状態の保存**: プレイヤーのステータスや所持品は、アクションのたびに「冒険の書」に保存されます。

## 4. 8方向移動の仕様
移動コマンドは以下の方向を受け付けます。
- `top` (上)
- `down` (下)
- `left` (左)
- `right` (右)
- `top-left` (左上)
- `top-right` (右上)
- `down-left` (左下)
- `down-right` (右下)

斜め移動は1ターンの行動として扱われ、グリッドベースの座標系において適切に処理されます。
